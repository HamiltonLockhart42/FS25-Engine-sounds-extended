# FS25 More Motor Sounds

This repository contains a Farming Simulator 25 script mod that adds a **More Motor Sounds** shop configuration to motorized vehicles.

## What it does
- Adds a `More Motor Sounds` vehicle configuration in the shop.
- Keeps the default game sound as option 1 (`Standard`).
- Adds multiple alternative engine sound sets (4-cyl, 6-cyl, 8-cyl, IHC, open pipe, blowoff).
- Automatically hides blowoff options in multiplayer.

## Installation
1. Zip the mod contents so that `modDesc.xml` is in the zip root.
2. Name the archive (for example): `FS25_MoreMotorSounds.zip`.
3. Copy the zip to your FS25 mods folder.
4. Start FS25 and activate the mod in your savegame.

## Notes
- This is an FS25 conversion of your FS22-style configurable engine-sound mod layout.
- Included `modDesc.xml` and Lua scripts are set up for FS25 (`descVersion="93"`).
